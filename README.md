# SpotWeb
Code for SpotWeb

Dependencies:
For SpotWeb to run, the following python libraries are needed:
a- boto3
b-flask
c-rpy2 (for the splines predictor)
d- pandas
e-flask_restful
f-pickle (for storing the output from the experiments)
g- cvxportfolio
h- cvxpy


For plotting:
a- seaborn
b-matplotlib
